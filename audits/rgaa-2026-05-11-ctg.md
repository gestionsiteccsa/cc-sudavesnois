## 2. Non-conformités détaillées

### Thème 1 : Images

**[NC] 1.1.1 — Logo CTG sans alternative textuelle**
- **Niveau :** A  
- **Gravité :** 🟠 Majeure  
- **Élément :** `<img src="{% static 'img/ctg/Jeudeloie.webp' %}" alt="Jeu de l'oie Sud-Avesnois" class="...">`  
- **Problème :** L'image est informative (affiche le visuel du jeu de l'oie CTG). L'alternative décrit le sujet mais ne précise pas le contexte (CTG).  
- **Correction :** `alt="Jeu de l'oie — Convention Territoriale Globale Sud-Avesnois"`

**[NC] 1.1.2 — Image document CTG sans alt**
- **Niveau :** A  
- **Gravité :** 🟠 Majeure  
- **Élément :** L'image de document/visuel CTG (vérifier si présente dans le template)  
- **Problème :** À vérifier dans le rendu réel — le template étant tronqué dans la sortie.

**[C] 1.2.1 — Motifs décoratifs**
- **Élément :** `<div aria-hidden="true">` sur les clip-paths  
- **Statut :** ✅ Conforme.

### Thème 8 : Éléments obligatoires

**[NC] 8.2.1 — Duplication de `<main>`**
- **Niveau :** A  
- **Gravité :** 🔴 Critique  
- **Élément :** `<main id="main-content">` dans le template + `<main>` dans `base.html`  
- **Problème :** Deux éléments `<main>` dans le DOM → HTML invalide, ambiguïté pour les AT.  
- **Correction :** Supprimer `<main id="main-content">` du template (garder celui de base.html).

**[NC] 8.2.2 — Attribut `aria-hidden` dupliqué sur SVGs**
- **Gravité :** 🟡 Mineure  
- **Élément :** Plusieurs SVGs avec `aria-hidden="true"` présent deux fois.  
- **Correction :** Supprimer la duplication.

### Thème 9 : Structuration

**[C] 9.1.1 — Hiérarchie de titres**
- **Élément :** H1 (bannière) → H2 (La convention / Les objectifs / Le comité / Les GAT)  
- **Statut :** ✅ Conforme — titres complets et explicites.

**[NC] 9.1.2 — Absence de `aria-label` sur les sections blocs**
- **Niveau :** A  
- **Gravité :** 🟡 Mineure  
- **Élément :** Les 4 blocs d'information sont des `<div>` sans `section` ni `aria-label`.  
- **Correction :** Soit les transformer en `<section>` avec `aria-labelledby`, soit ajouter un `aria-label`.

```html
<!-- APRÈS -->
<section class="bg-white dark:bg-gray-800 rounded-lg ... border-l-4 border-primary"
         aria-labelledby="heading-convention">
    <div class="flex items-center mb-4">
        ...
        <h2 id="heading-convention" class="text-2xl font-bold ...">La convention</h2>
    </div>
    ...
</section>
```

### Thème 10 : Présentation

**[NC] 10.7.1 — Contraste texte dans les encarts de couleur**
- **Niveau :** AA  
- **Gravité :** 🟠 Majeure  
- **Élément :** Cartes GAT : `text-gray-700 dark:text-gray-300` sur `bg-green-50 dark:bg-green-900/20`  
- **Problème :** En mode clair : `#374151` (gray-700) sur `#F0FDF4` (green-50) → ratio ~5.5:1 ✅ Conforme AA.  
  En mode sombre : `#D1D5DB` (gray-300) sur fond avec opacité 20% de `#14532D` (green-900) → le fond réel dépend du parent. **À vérifier.**  
- **Correction :** Si le fond sombre est trop proche de la couleur du texte, remplacer `text-gray-300` par `text-gray-200`.
